# 电竞平台可视化控制台 (Admin Console) 使用说明

本控制台采用了类似 WordPress 的现代、简约布局，旨在为管理员提供一站式、可视化的运营能力。

## 1. 进入方式
*   **入口 URL**：`http://localhost:5173/admin`
*   **权限要求**：目前拦截器已放行管理路径，但建议在生产环境将用户表 `role` 字段设为 `ROLE_ADMIN`。

## 2. 核心功能模块

### 🚀 首页看板 (Dashboard)
*   **实时数据**：直观展示全站用户量、战队数、约赛场次、发帖量。
*   **可视化图表**：采用 Echarts 渲染游戏项目占比分布图，以及用户增长趋势。

### 👥 权限管理
*   **用户列表**：查看所有注册用户的高校信息与认证状态。
*   **审核/冻结**：一键审核新用户或冻结违规账号，操作立即在数据库生效。

### ⚙️ 系统设置 (WordPress 风格)
*   **基础配置**：直接在 Web 界面修改网站标题（Site Name）、备案号。
*   **业务开关**：可视化开启或关闭“新用户注册”、“约赛自动审核”等核心逻辑。
*   **资源替换**：支持通过图片 URL 一键更换首页 Banner 背景图。

### 📢 内容发布
*   **公告管理**：支持公告的发布、编辑、实时下架与浏览量查看。
*   **留言处理**：（开发中）统一查收用户反馈并进行回复。

### 🏆 业务管理
*   **战队/约赛/社区**：提供全站业务数据的监控与逻辑删除能力。

---

## 3. 技术联动说明
1.  **接口联动**：所有控制台操作均调用 `/api/**/admin` 前缀接口，后端 `LoginInterceptor` 已对此类高权操作预留扩展。
2.  **配置驱动**：`SysConfig` 模块实现了“配置即数据”，前端展示逻辑会根据控制台的配置实时改变。
3.  **响应式设计**：后台采用经典的左侧菜单+右侧内容区，支持侧边栏折叠，适配不同分辨率的显示器。
